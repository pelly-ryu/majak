About AlphaJong Review System

This review system analyzes mahjong positions and provides educational feedback to help players learn optimal strategies. It uses the same AI engine as the AlphaJong userscript bot.

How to Use:

  1. Load a Position: Enter a debug string, use an example, or try a teaching scenario
  2. Get AI Analysis: Click "Get AI Recommendation" to see what the AI would discard
  3. Compare Moves: Enter your choice to compare with the AI and get feedback
  4. Learn Concepts: Use teaching scenarios to practice specific mahjong concepts
  5. Build Custom Hands: Use the hand builder to create your own positions

Debug String Format:

dora|hand|calls0|calls1|calls2|calls3|discards0|discards1|discards2|discards3|riichi|seatWind|roundWind|tilesLeft

Tiles are in format: 123m456p789s1z (m=man, p=pin, s=sou, z=honors)
